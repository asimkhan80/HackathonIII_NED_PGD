# LearnFlow - AI-Powered Learning Platform

**Hackathon III: Reusable Intelligence and Cloud-Native Mastery**

## Overview

LearnFlow is a cloud-native, AI-powered learning platform that demonstrates **Agent-First Development**. All infrastructure and application code was generated by AI agents using reusable Skills.

## Architecture

The full application runs on Kubernetes with:

- **Frontend:** Next.js 14 with TypeScript
- **Backend:** FastAPI with Dapr integration
- **Messaging:** Apache Kafka via Strimzi (KRaft mode)
- **State Store:** PostgreSQL
- **Service Mesh:** Dapr for pub/sub and state management
- **Orchestration:** Kubernetes (Kind cluster)

## This Demo

This Gradio demo simulates the core functionality:

1. **State Store** - Save and retrieve state (simulates PostgreSQL via Dapr)
2. **Pub/Sub** - Publish messages to topics (simulates Kafka via Dapr)
3. **Architecture** - View the cloud-native architecture diagram
4. **Skills** - Learn about reusable AI agent skills

## Local Deployment

To run the full stack locally:
